## Библиотеки (модули) Python
 
Библиотеки Python — это наборы готовых модулей и функций, которые расширяют возможности языка, облегчая разработку и сокращая время на написание кода. Наборы готовых решений, которые содержат функции, классы и модули для упрощения разработки программ. Они позволяют не изобретать заново типовые методы, а использовать уже написанные и проверенные на практике фрагменты кода. <br><br>
Представь, что ты написал небольшую программу для получения прогноза погоды с открытого API и последующего анализа данных. Например, она отправляет запросы на сервер погоды, получает информацию о температуре, влажности, скорости ветра и выдаёт краткий прогноз. Ты решил оформить эту программу как библиотеку, чтобы в будущем легко использовать её в других проектах. <br><br>
Когда программист использует библиотеку, он подключает к своей программе заранее созданные инструменты, которые выполняют сложные или повторяющиеся задачи. Например, если нужно выполнить математические расчёты, взаимодействовать с базой данных или обработать изображение, нет необходимости писать эти функции с нуля — библиотека предоставляет готовые механизмы для этого. <br><br>
Главное преимущество библиотек — это ускорение разработки и повышение надёжности кода, ведь они создаются и поддерживаются сообществом или профессиональными разработчиками, которые уделяют внимание безопасности, эффективности и функциональности решений. <br><br>
[Официальная документация](https://docs.python.org/3/tutorial/modules.html) по модулям Python.

## Как выглядит модуль

<img src="https://github.com/TeachKait20/NoneCode/blob/main/modules+py+git/Study_Work.gif?raw=true" width="400">

Допустим, вы создали файл с функцией для сложения чисел:
```python
# файл math_library.py

def add_numbers(a, b):
    return a + b
```
Теперь в другой программе вы можете импортировать свою функцию и использовать её для сложения чисел:
```python
# файл main.py

from math_library import add_numbers

def main():
    result = add_numbers(5, 3)
    print(f"Result of addition: {result}")

# Вызов функции
main()
```

## Встроенные модули и внешние

<img src="https://github.com/TeachKait20/NoneCode/blob/main/modules+py+git/New%20Products%20Every%20Day.gif?raw=true" width="400">

Различия между встроенными и внешними модулями:

* Встроенные модули всегда доступны и не требуют установки.
* Внешние модули требуют предварительной установки через `pip`.

### Встроенные модули

Это библиотеки, которые идут в комплекте с интерпретатором Python. Для их использования не требуется установка, достаточно просто импортировать их в коде. Примеры встроенных модулей: math, os, sys, datetime.<br><br>
Чтобы использовать встроенный модуль, нужно просто импортировать его с помощью ключевого слова import или из него отдельные функции с `from ... import ....` <br><br>
Каждая библиотека предназначена для своей области применения, и поэтому набор методов в ней подстраивается под эти цели. <br><br>
Вызов встроенного модуля целиком:
```python
import math

# Используем функцию sqrt из модуля math
result = math.sqrt(16)
print(result)  # Вывод: 4.0
```
Вызов только конкретной функции из модуля:
```python
from math import sqrt

# Используем функцию sqrt напрямую, без math. перед ней
result = sqrt(16)
print(result)  # Вывод: 4.0
```
### Внешние модули

Это те библиотеки, которые не входят в стандартную поставку Python. Чтобы их использовать, необходимо сначала установить их через менеджер пакетов `pip`, а затем импортировать в коде. <br><br>
`pip` — это пакетный менеджер для Python, который используется для установки, управления и обновления внешних библиотек (или пакетов). С его помощью можно легко находить, загружать и устанавливать необходимые сторонние модули, которые не входят в стандартную библиотеку Python. <br><br>
Установка внешнего модуля. Обычно внешние модули устанавливаются через командную строку с помощью `pip`. Например, для установки библиотеки requests нужно выполнить: <br><br>
`pip install requests` <br><br>
Импорт внешнего модуля. После установки ты можешь использовать его в своей программе, как и встроенные модули. <br><br>
Использование модуля requests для работы с HTTP-запросами:
```python
import requests

response = requests.get('https://api.github.com')
print(response.status_code)  # Вывод: 200 (ОК)
```
Импорт только нужных элементов из внешнего модуля:
```python
from requests import get

response = get('https://api.github.com')
print(response.status_code)  # Вывод: 200
```
## Основные категории и примеры популярных библиотек

<img src="https://github.com/TeachKait20/NoneCode/blob/main/modules+py+git/add.gif?raw=true">

**1.** Стандартная библиотека Python

* os — работа с файловой системой, взаимодействие с операционной системой.
* sys — управление аргументами командной строки, работа с интерпретатором Python.
* math — математические операции (тригонометрия, логарифмы, факториалы).
* datetime — работа с датами и временем.
* json — работа с JSON форматами (сериализация/десериализация данных).
* random - псевдослучайные события. 

**2.** Библиотеки для работы с данными
   
* NumPy — библиотека для работы с многомерными массивами и матричными вычислениями.
* Pandas — библиотека для анализа данных, работа с табличными данными (DataFrame).
* Matplotlib и Seaborn — библиотеки для визуализации данных, создания графиков и диаграмм.
* SciPy — библиотека для научных и технических вычислений.

**3.** Веб-разработка

* Django — фреймворк для создания веб-приложений, включает инструменты для администрирования, безопасности, работы с базами данных.
* Flask — легковесный веб-фреймворк для создания веб-сервисов и REST API.
* FastAPI — современный фреймворк для быстрого создания API, поддерживает асинхронное программирование.

**4.** Машинное обучение и искусственный интеллект
   
* TensorFlow и PyTorch — библиотеки для создания нейронных сетей, работы с большими данными и машинного обучения.
* Scikit-learn — библиотека для машинного обучения, предоставляет инструменты для классификации, регрессии и кластеризации.
* Keras — высокоуровневая библиотека для работы с нейронными сетями, поверх TensorFlow.

**5.** Автоматизация и работа с вебом
   
* Requests — библиотека для отправки HTTP-запросов, взаимодействия с API.
* BeautifulSoup и Scrapy — инструменты для веб-скрапинга, извлечения данных с веб-страниц.
* Selenium — для автоматизации браузеров, тестирования и взаимодействия с веб-страницами.

**6.** Работа с базами данных
   
* SQLAlchemy — библиотека для работы с базами данных, предоставляет ORM (Object-Relational Mapping) для взаимодействия с SQL-базами.
* SQLite — встроенная база данных, для работы с которой в Python используется модуль sqlite3.

**7.** Библиотеки для тестирования
    
* unittest — встроенная библиотека для написания тестов.
* pytest — более гибкий инструмент для тестирования, поддерживает параметризацию, плагины.
* mock — для создания mock-объектов при тестировании, эмулирования поведения других модулей.

**8.** Работа с изображениями и видео
    
* Pillow — библиотека для работы с изображениями, поддержка форматов JPEG, PNG, GIF и др.
* OpenCV — библиотека для обработки изображений и видео, используемая в задачах компьютерного зрения.

**9.** Работа с сетью и протоколами
    
* Socket — стандартная библиотека для работы с сетевыми соединениями.
* Twisted — библиотека для асинхронной обработки сетевых событий.
* Paramiko — для работы с SSH, SFTP.

**10.** Асинхронное программирование
    
* asyncio — стандартная библиотека для создания асинхронных программ.
* aiohttp — асинхронная библиотека для работы с HTTP-запросами.

Каждая из этих библиотек имеет свои уникальные возможности и удобства для различных задач разработки. Это лишь малая часть.

## Примеры работы с модулями

<img src="https://github.com/TeachKait20/NoneCode/blob/main/modules+py+git/tools.gif?raw=true" width="400">

Рассмотрим несколько популярных библиотек Python и примеры того, как их можно вызывать и использовать в коде.

### Пример 1
`random` — это встроенная библиотека Python, которая используется для работы с случайными числами и выборками. <br><br>
`random.randint(num1, num2)` - Этот метод генерирует случайное целое число в диапазоне от num1 до num2, включая обе границы. Каждый раз при вызове программа будет выдавать новое случайное число. <br><br>
Напишем программу с генерацией случайного числа в нескольких примерах. <br>

**1. Полный импорт библиотеки**
```python
import random

random_number = random.randint(-100, 100)
print(random_number)
```
* Здесь импортируется вся библиотека `random`. Для доступа к любой функции из этой библиотеки нужно указывать её через точку, как `random.randint()`.
* Это стандартный и наиболее читаемый способ импорта, особенно когда нужно использовать несколько функций из модуля.
  
**2. Импорт конкретной функции**
```python
from random import randint

random_number = randint(-100, 100)
print(random_number)
```
* Импортируется только одна конкретная функция `randint` из модуля `random`.
* При вызове этой функции не нужно использовать префикс модуля, можно сразу писать `randint()`.

**3. Импорт всех функций из модуля**
```python
from random import *

random_number = randint(-100, 100)
print(random_number)
```
* Импортируются все функции из модуля `random`. Все функции становятся доступны напрямую, без указания имени модуля.

В этих трёх примерах показаны разные способы импорта библиотеки random, и все они позволяют выполнить одну и ту же задачу — сгенерировать случайное число в диапазоне от -100 до 100. Их можно применять и для других модулей.

### Пример 2

`colorama` — это внешняя библиотека, которая используется для вывода цветного текста в консоли. <br><br>
Сначала нужно установить библиотеку colorama, если она ещё не установлена: <br><br>
![image](https://github.com/user-attachments/assets/50c1dff4-e14c-4cae-9536-37f06a27d9d2)
<br>
Пример использования:
```python
import colorama

# Инициализация библиотеки, чтобы цвета автоматически сбрасывались после вывода
colorama.init(autoreset=True)

# Вывод текста с разными цветами
print(colorama.Fore.GREEN + "Colored text")
print(colorama.Fore.CYAN + "More colored text")
print(colorama.Fore.BLUE + "Even more colored text")
```
* `colorama.init(autoreset=True)` - Инициализация библиотеки, которая активирует поддержку цветов и сбрасывает цвет после каждого вызова, чтобы следующий текст выводился в стандартном цвете. Это удобно, чтобы не нужно было вручную сбрасывать цвет после каждой строки.
* `colorama.Fore` - Этот атрибут используется для изменения цвета переднего плана текста. Например, `colorama.Fore.GREEN` устанавливает текст зелёного цвета, а `colorama.Fore.CYAN` и `colorama.Fore.BLUE` — соответственно, для цвета циан и синего.
* `print()` - Цветной текст выводится на экран благодаря добавлению цветового кода перед строкой.
